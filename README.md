# 从一次提交到一次协作：Git 与 GitHub 开源项目实践入门

欢迎参加本次 Git 与 GitHub 实践讲座。本仓库是讲座配套练习仓库，参与者将通过一次完整的提交（commit）到拉取请求（Pull Request）流程，体验开源项目的协作方式。

## 仓库结构

```
├── README.md          # 本文件，项目说明
├── CONTRIBUTING.md    # 贡献规范说明
├── LICENSE            # 许可证（MIT）
├── members/           # 在此添加个人介绍文件
├── docs/              # 课堂笔记文档
└── src/               # 代码示例
```

## 快速开始

1. 复刻（fork）本仓库到你的 GitHub 账号
2. 克隆（clone）到本地：`git clone https://github.com/<你的用户名>/git-show.git`
3. 配置身份，完成两次提交
4. 推送（push）到 GitHub 并发起拉取请求（Pull Request）

详细步骤请阅读 [CONTRIBUTING.md](CONTRIBUTING.md)。
