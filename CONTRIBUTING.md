# 贡献规范

感谢参与本次实践！请按以下步骤完成贡献。

## 任务要求

1. 在 `members/` 目录下新增一个以你的 GitHub 用户名命名的 Markdown 文件，例如 `members/your-username.md`
2. 在 `docs/git-notes.md` 中补充一句你今天的收获
3. 形成至少两次提交（commit）
4. 推送（push）到你的复刻仓库（fork）
5. 向本仓库发起拉取请求（Pull Request）

## 提交信息规范

- 使用英文或中文清晰说明改动内容
- 格式：`<type>: <description>`
- 示例：`add profile: your-username`、`update git notes`

## 分支命名

本次实践统一使用 `main` 分支。

## PR 标题格式

`Add profile: <你的用户名>`

## 文件规范

### members/<用户名>.md

```markdown
# 姓名

- **学院**：你的学院
- **兴趣方向**：你感兴趣的领域
- **GitHub**：[用户名](https://github.com/用户名)
```

### docs/git-notes.md

在文件末尾新增一行，格式为 `- [你的用户名] 你的收获`。

## 检查清单

提交 PR 前请确认：

- [ ] 新增了 `members/<用户名>.md`
- [ ] 更新了 `docs/git-notes.md`
- [ ] 包含至少两次提交（commit）
- [ ] PR 标题格式正确
- [ ] PR 描述填写完整
